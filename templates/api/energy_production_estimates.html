Annual State Energy Production Estimates

Data Description

Annual estimates of energy production by state.

Format of API Call

http://data.nationalpriorities.org/api/energy_production_estimates/list.<format>?...

Allowed Keys: 'msn', 'year', 'state'

Example: http://data.nationalpriorities.org/api/energy_production_estimates/list.json?state=WY&year=2007

Returned Fields

  • id: Internal Database ID (may change, do not rely on)
  • msn: Code to Describe Energy Source, Sector, and Type of Data (list of msn's)
  • year
  • state: U.S. State
  • value: Energy Production Value

Source

Energy Information Administration (EIA), State Energy Data System (SEDS)

Notes

  • Computation of ethanol estimates isn't reliable
  • 'value' units depend on MSN (list of MSN's)
